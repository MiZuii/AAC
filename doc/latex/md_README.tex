Little simpley library bringing joy to our dull lifes by creating cute images of ascii frogs and other heart warming things.\hypertarget{md_README_autotoc_md2}{}\doxysubsection{Docs}\label{md_README_autotoc_md2}
For documentation install Doxygen with Graphviz for website version and for pdf version install latex compiler. After building project run {\ttfamily make doc} to generate website documentation and {\ttfamily make man} for pdf documentation.

Installation of latex


\begin{DoxyCode}{0}
